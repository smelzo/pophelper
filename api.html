Options

Most options can been setted via data- attribute

Option Name Data attribute Type Default description
activeClass data-active-class String 'active' Css class for element when Popover is showed
tip data-tip String '' if setted the text is used to show a tooltip on element over
contentTarget data-content-target String '' selector or element used as popover's content data-content-target="#my-help-content"
href data-href String '' external document URL (but inside same domain) used as popover's content data-href="http://www.mysyte/help/doc1.html"
popTarget data-pop-target String element selector or element in relation to which popover is open ie. data-pop-target="#an-element"
html - Boolean true Insert html into the popover. If false, jquery's text method will be used to insert content into the dom.
placement data-placement String 'bottom' side of popover in relation to element top|left|bottom|right
width data-width Number 'auto' width of popover
maxWidth data-max-width Number 'auto' max width of popover
minWidth data-max-width Number 'auto' max width of popover
height data-height Number 'auto' height of popover
maxHeight data-max-height Number 'auto' max height of popover
minHeight data-max-height Number 'auto' max height of popover
