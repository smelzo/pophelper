Usage

  1. Insert CSS and JS reference in the HEAD of document after jQuery and Bootstrap 
    <head>
...
<link href="path_to_css/bootstrap.css" rel="stylesheet">
<link href="path_to_css/pophelper.css" rel="stylesheet">

...
<script src="path_to_scripts/jquery.js"></script>
<script src="path_to_scripts/bootstrap.js"></script>
<script src="path_to_scripts/jquery.pophelper.js"></script>
</head>
    if you prefer you can put javascript files reference also before tag </body>
  2. Insert the element that will activate pop 
        <a href="#" data-toggle="pophelper">Press for help</a>
  3. In javascript code apply plugin 
        $(function (){
        $('[data-toggle="pophelper"]').pophelper();
    })
  4. You can set option in element code by data attributes… 
        <a href="#" data-toggle="pophelper" data-content-target="#myHelp" data-title="my help" data-placement="top">Press for help</a>
    …OR …
  5. … via javascript configuration object 
        $(function (){
        $('[data-toggle="pophelper"]').pophelper({
            contentTarget: "#myHelp"
            , title:"my help"
            , placement:"top"
        });
    })

Options

Most options can been setted via data- attribute

Option Name Data attribute Type Default description
activeClass data-active-class String 'active' Css class for element when Popover is showed
tip data-tip String '' if setted the text is used to show a tooltip on element over
contentTarget data-content-target String '' selector or element used as popover's content data-content-target="#my-help-content"
cloneContentTarget data-clone-content-target Boolean false if setted contentTarget clone element used as popover's content
href data-href String '' external document URL (but inside same domain) used as popover's content data-href="http://www.mysyte/help/doc1.html"
popTarget data-pop-target String element selector or element in relation to which popover is open ie. data-pop-target="#an-element"
html - Boolean true Insert html into the popover. If false, jquery's text method will be used to insert content into the dom.
placement data-placement String 'bottom' side of popover in relation to element top|left|bottom|right
width data-width Number 'auto' width of popover
maxWidth data-max-width Number 'auto' max width of popover
minWidth data-max-width Number 'auto' max width of popover
height data-height Number 'auto' height of popover
maxHeight data-max-height Number 'auto' max height of popover
minHeight data-max-height Number 'auto' max height of popover

Methods

$().pophelper(options)

Initializes pophelpers for an element collection.

.pophelper('show')

Reveals an elements pophelper.

$('#element').pophelper('show')
help 
<a href="#" id="method-show" data-content="[inline content...]" data-title="help">help</a>
<script type="text/javascript">
    jQuery(function (){
        $('#method-show').pophelper({placement:'right',width:300}).delay(2000).pophelper('show');
    })
</script>

.pophelper('hide')

Hides an elements popover.

$('#element').pophelper('hide')

.pophelper('toggle')

Toggles an elements popover.

$('#element').pophelper('toggle')
help click for toggle pop 
<a href="#" id="method-toggle-popcall" data-content="This help is opened programmatically using show method" data-title="help">help</a>
<a href="#" id="method-toggle-toggler" class="btn pull-right pophelper-toggle">click for toggle pop</a>
...
<script type="text/javascript">
    jQuery(function (){
        var el = $('#method-toggle-popcall').pophelper({placement:'left',width:200});
        $('#method-toggle-toggler').click(function (event){
            event.preventDefault();
            el.pophelper('toggle');    
        });
    })
</script>
Heads up! The element that call .pophelper('toggle') must have pophelper-toggle class for work properly.
<a href="#" class="pophelper-toggle">Toggle</a>

.pophelper('destroy')

Hides and destroys an element's pophelper.

$('#element').pophelper('destroy')

Events

Shared with Bootstrap's Tooltip

Event Description
show This event fires immediately when the show instance method is called.
shown This event is fired when the modal has been made visible to the user (will wait for css transitions to complete).
hide This event is fired immediately when the hide instance method has been called.
hidden This event is fired when the modal has finished being hidden from the user (will wait for css transitions to complete).

PopHelper's events

Event Description
pophelper.before.setup This event fires before the instance setup.
pophelper.after.setup This event fires after the instance setup.
help
Log:
<script type="text/javascript">
    jQuery(function (){
        var _log = function (txt){
            //...
        }
        $('#events-test').on('pophelper.before.setup',function (){
            _log('before setup');
        });
        $('#events-test').on('pophelper.after.setup',function (){
            _log('after setup');
        });
        $('#events-test').on('show',function (){
            _log('show pop');
        });
        $('#events-test').on('shown',function (){
            _log('shown pop');
        });
        $('#events-test').on('hide',function (){
            _log('hide pop');
        });
        $('#events-test').on('hidden',function (){
            _log('hidden pop');
        });
        var el = $('#events-test').pophelper({placement:'left',width:200});
        
    })
</script>
Warning! pophelper.before.setup and pophelper.after.setup must be binded before to apply plugin to element.
