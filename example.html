Programmatically Pop Helper

You can call $(...).pophelper from javascript 

      <button id="by_code" class="btn btn-inverse" type="button">
        <i class="icon-question-sign"></i>
      </button>
      <script type="text/javascript">
        $(function (){
          $('#by_code').pophelper({
            content : '<h4>I am the content</h4>'
            , title : 'Programmatically Pop Helper'
            , minWidth : 400
            , placement : 'right'
          });
        });
      </script>

Load remote content

You can use as content of your Popover an external document (in same domain) just set the url in data-href of element or in href option

Targetting popover

open near another element than toggler button

You can set option popTarget or data-attribute data-pop-target to link Popover to an another element than help button.

In example below you see that Popover is attached to input rather than a

Legend 
<label>Label
  <a href="#" class="pophelper-test" data-pop-target="#iam_the_target" data-href="test/help.html">
    <i class="icon-question-sign"></i>
  </a>
</label>
<input id="iam_the_target" type="text" placeholder="Type something…">
